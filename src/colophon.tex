\lettrine[lraise=-0.03,loversize=0.08]{本}{书}由 \urlref{https://hmemcpy.com}{Igal Tabachnik} 编译完成。他将 Bartosz Milewski 的原始文本转换为 \LaTeX{} 格式，首先使用 \urlref{https://mercury.postlight.com/web-parser/}{Mercury Web Parser} 抓取原始 WordPress 博客文章以获取干净的 HTML 内容，然后使用 \urlref{https://www.crummy.com/software/BeautifulSoup/}{Beautiful Soup} 进行修改和调整，最后使用 \urlref{https://pandoc.org/}{Pandoc} 转换为 \LaTeX{}。

原书的版式设计和排版由 Andres Raba 完成。语法高亮使用 \urlref{https://github.com/hugomaiavieira/pygments-style-github}{Hugo Maia Vieira} 的 "GitHub" 风格 Pygments。
